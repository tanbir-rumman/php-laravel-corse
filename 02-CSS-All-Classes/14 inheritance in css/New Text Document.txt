class 14 inharitance in css কি হয় ?

HTML কে সুন্দর করে সাজানোর জন্য HTML এর প্রতিটি element কে সুন্দর ডিজাইন করা হয়ে থাকে css এর মধ্যমে

inharitance in css কি হয়

উত্তরঃ-  এক কথায় যধি বলি তাহলে css এ অনেক div থাকে , 
একটি div এর মধ্যে মধ্যে যে অনেক content থাকে  আর প্রতিটি div কে select করার জন্য আমরা অনেক উপায় জানি 

এখন একটি div কে class এর মাধ্যমে select করে তাকে বলা হলো যে color : red;
এখন এর মধ্যে থাকা সকল element গুলোই কিন্তু color red হয়ে যাবে
উক্ত div এর মধ্যে থাকা element গুলোকে এখানে আলাদা করে বলার আর কিছুই নেই , 
তবে এখানে সকল উক্ত div এ থাকা  css কিন্তু সেই color টা নাও নিতে পারে
যেমন a tag এই elemen কে আলাদা করে color দিতে হয়ে।

যখন আমরা একটি div কে একটি ডিজাইন দিয়ে দেই তখন তার মধ্যের content সেখান থেকে 
 ডিজাইন নিয়ে নেয় আর এটাকেই বলে inharitance css
