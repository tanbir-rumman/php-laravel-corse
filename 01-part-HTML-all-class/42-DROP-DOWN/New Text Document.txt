class 42
==========================================
ক্লাস 42 drop down menu কি হয় ?
==========================================

উত্তরঃ drop down menu এটি একটি  HTML ট্যাগ,  যার কাজ মেনু সিলেক্টর
এটির মাধ্যমে একটি মেনুকে drop down আকারে পেজ এ দেখানো

optgroup কি ?

optgroup এটির মাধ্যমে একটি মেনুর মধ্যে অন্য আরো একটি সাব মেনুর মত তৈরি করা

select কি ?

উত্তরঃ select এটি একটি অ্যাট্রিবিউট এটির মাধ্যমে একটি মেনুকে ডিফল্ট ভাবে সিলেক্ট করা হয়
